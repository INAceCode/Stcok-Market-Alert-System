 Stock Insight Bot

A Python-based tool designed to monitor financial markets, analyze historical data, and detect significant price movements.

## Features

* Fetches real-time and historical stock data using Yahoo Finance API
* Calculates daily performance metrics
* Detects significant price movements based on custom thresholds
* Stores data in a SQLite database
* Generates automated Excel reports

## Technologies

* Python
* Pandas
* SQLite
* yFinance

##  How It Works

1. Retrieves historical data for selected assets
2. Calculates daily price variations
3. Identifies positive/negative trends
4. Triggers alerts for significant movements
5. Stores results and generates reports
